Task 1: Enable and Encrypt Passwords

Enable passwords and ensure they are encrypted in the configuration file to prevent them from being easily visible.

enable
configure terminal
enable secret [your-password]
service password-encryption

Flag 1: Use show running-config to verify the encryption is enabled.

Task 2: Create VLANs

Create VLANs 10, 20, and 30 for different departments in the company (e.g., HR, IT, and Finance).

vlan 10
name HR
vlan 20
name IT
vlan 30
name Finance
exit

Flag 2: Verify the VLAN creation using show vlan.

Task 3: Configure Switchport Access

Assign the access ports on the switch to their respective VLANs.

interface range FastEthernet0/1 - 10
switchport mode access
switchport access vlan 10

Flag 3: Verify the configuration using show interfaces switchport.

Task 4: Trunk Configuration and Access Control

Configure trunk ports to allow specific VLANs and deny others.

interface GigabitEthernet0/1
switchport mode trunk
switchport trunk allowed vlan 10,20
switchport trunk denied vlan 30

Flag 4: Verify the trunk configuration using show interfaces trunk.

Task 5: Implement SNMP for Network Monitoring

Enable and configure SNMP version 2c with a community string "public" for read-only access.

snmp-server community public RO

Flag 5: Use show snmp to verify SNMP configuration.

Task 6: Implement Port Security

Secure port FastEthernet0/5 by limiting the number of allowed MAC addresses to 1 and setting up violation action as shutdown.

interface FastEthernet0/5
switchport mode access
switchport port-security
switchport port-security maximum 1
switchport port-security violation shutdown

Flag 6: Verify port security using show port-security interface FastEthernet0/5.

Task 7: Prevent Brute Force Attacks on Password

Set up login security to prevent brute force attacks by limiting the number of login attempts and introducing a login delay.

line vty 0 4
login block-for 120 attempts 3 within 60

Flag 7: Verify the login settings using show login.
