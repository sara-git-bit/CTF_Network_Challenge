Capture the Flag (CTF) Challenge: Network Troubleshooting and Configuration

Welcome to the CTF Challenge! You are the network administrator responsible for configuring and securing the network. Complete each task and enter the correct flag to proceed.

Scenario:

Your company has experienced various network issues and security concerns. Your job is to troubleshoot, configure, and secure the network devices. For each task, after successfully configuring the network, enter the flag that confirms you've completed it correctly.

Task 1: Enable and Encrypt Passwords

Enable passwords and ensure they are encrypted in the configuration file.

enable
configure terminal
enable secret [your-password]
service password-encryption

Flag 1: Use show running-config to verify the encryption is enabled.

Task 2: Create VLANs

Create VLANs 10, 20, and 30 for different departments (HR, IT, Finance).

vlan 10
name HR
vlan 20
name IT
vlan 30
name Finance
exit

Flag 2: Verify VLANs using show vlan.

Task 3: Configure Switchport Access

Assign the access ports on the switch to their respective VLANs.

interface range FastEthernet0/1 - 10
switchport mode access
switchport access vlan 10

Flag 3: Verify using show interfaces switchport.

Task 4: Trunk Configuration and Access Control

Configure trunk ports to allow specific VLANs and deny others.

interface GigabitEthernet0/1
switchport mode trunk
switchport trunk allowed vlan 10,20
switchport trunk denied vlan 30

Flag 4: Verify using show interfaces trunk.

Task 5: Implement SNMP for Network Monitoring

Enable SNMP version 2c with a community string "public" for read-only access.

snmp-server community public RO

Flag 5: Use show snmp to verify.

Task 6: Implement Port Security

Secure port FastEthernet0/5 by limiting the number of allowed MAC addresses to 1 and setting the violation action to shutdown.

interface FastEthernet0/5
switchport mode access
switchport port-security
switchport port-security maximum 1
switchport port-security violation shutdown

Flag 6: Verify using show port-security interface FastEthernet0/5.

Task 7: Prevent Brute Force Attacks on Password

Set up login security to prevent brute force attacks by limiting the number of login attempts.

line vty 0 4
login block-for 120 attempts 3 within 60

Flag 7: Verify using show login.

Good luck securing the network!
