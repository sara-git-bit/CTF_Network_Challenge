Capture the Flag (CTF) Challenge

Network Troubleshooting and Configuration

Welcome to the Network CTF Challenge! You are tasked with configuring and securing a network. Complete each task and enter the correct flag to unlock the next challenge. Good luck!

Task 1: Enable and Encrypt Passwords

Enable passwords and ensure they are encrypted in the configuration file.

enable password [your-password]

Flag 1: Use correct to verify the encryption.

Task 2: Create VLANs

Create VLANs for the different departments: HR (VLAN 10), IT (VLAN 20), and Finance (VLAN 30).

vlan 10
name HR
vlan 20
name IT
vlan 30
name Finance
exit

Flag 2: Use correct to configure all the vlans.

Task 3: Configure Switchport Access

Assign the access ports to the appropriate VLANs. Assign port f0/2 to f0/3 to vlan 10, port f0/4 to f0/5 to vlan 20 and port f0/6 to f0/7 to vlan 30 

interface range f0/1-2 , interface range f0/3-4, interface range f0/5-6
switchport mode [type of mode]

Flag 3: Use correct to configure the access.

Task 4: Trunk Configuration and Access Control

Configure trunk ports to allow specific VLANs and deny others.

interface GigabitEthernet0/1

Flag 4: Use correct to configure trunk.

Task 5: Implement SNMP for Network Monitoring

Enable SNMP version 2c with a community string "public" for read-only access.

snmp-server

Flag 5: Use correct to configure the SNMP.

Task 6: Implement Port Security

Secure port FastEthernet0/5 by limiting the number of allowed MAC addresses and setting violation action to shutdown.

interface FastEthernet0/5

Flag 6: Use correct to configure port-security.

Task 7: Prevent Brute Force Attacks

Configure login security to prevent brute force attacks by limiting login attempts and adding a login delay.

line vty 0 4

Flag 7: Use correct to prevent the brute force attacks.
